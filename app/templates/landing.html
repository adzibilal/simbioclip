Fitur

Semua yang kamu butuhkan

Dari download sampai render — satu pipeline otomatis tanpa ribet.

URL & Upload

Download dari YouTube dll via yt-dlp, atau upload file langsung. Preview metadata sebelum proses.

Speech-to-Text

Transkrip dengan timestamp per kata. Pilih mode API router (cepat) atau faster-whisper lokal (offline).

Deteksi Momen AI

LLM otomatis pilih momen terbaik dengan skor + hook + judul. Funny-first: momen lucu diprioritaskan.

Banyak Layout

Center Crop, Talking Head, Podcast, Split Cam, BG Blur — dengan rasio 9:16, 1:1, 4:5, 16:9.

Gaya Caption

Bold Pop, Karaoke, Neon, Minimal, Podcast — ganti gaya subtitle per-clip tanpa re-render penuh.

Edit Per-Clip

Rerender layout berbeda, trim per-frame, thumbnail cover custom — semua tanpa reproses penuh.

Dashboard Web

Pantau progres job live (HTMX + WebSocket), atur antrean, download clip & thumbnail dari browser.

LLM Router

Gratis — pakai Groq, OpenRouter, Google Gemini. Rotasi & fallback otomatis antar 9 endpoint.

API Lengkap

REST API untuk job, preview, download, trim, rerender, dan thumbnail. Integrasikan dengan workflow kamu.

Pipeline

Cara kerjanya

Setiap tahap bisa diulang secara independen. Transparan penuh.

1

Submit Video

Tempel URL YouTube atau upload file video. Atur jumlah clip, layout, dan preferensi lainnya.

2

AI Processing

Transkrip (STT) → deteksi momen (LLM) → klasifikasi konten → diarisasi pembicara. Semua otomatis.

3

Render Shorts

FFmpeg potong segmen, reframe ke 9:16 (atau rasio lain), burn caption — semua otomatis.

4

Download & Edit

Preview clip, edit hook/title, trim per-frame, ganti layout — lalu download langsung.

Arsitektur

VPS hanya kerja mekanis

STT & LLM di-offload ke router gratis. VPS cuma download, potong, render.

Known Issue

YouTube Cookie Bug

YouTube makin sering memblokir IP server. Ini solusi sementaranya.

Masalah

YouTube sering menampilkan "Sign in to confirm you're not a bot" untuk IP server/VPS. Akibatnya download video gagal. Video dengan batasan usia juga butuh autentikasi.

Solusi Sementara

Export cookies dari browser yang sudah login ke YouTube, lalu upload ke dashboard.

Langkah-langkah:

1

Pasang ekstensi "Get cookies.txt LOCALLY" dari Chrome Web Store.

2

Login ke youtube.com di Chrome, klik ekstensi → Export.

3

File cookies.txt akan terunduh. Salin ke root proyek:

# dari laptop ke VPS
scp ~/Downloads/cookies.txt user@ip-vps-anda:/path/simbioclip/cookies.txt

Catatan Penting

  • Cookies punya masa berlaku — kalau download mulai gagal, export ulang.
  • File bisa diupload langsung lewat Dashboard → YouTube FAB (tombol floating).
  • Cookies otomatis di-.gitignore — jangan pernah commit!
  • Opsional untuk video publik, wajib untuk video age-restricted.
